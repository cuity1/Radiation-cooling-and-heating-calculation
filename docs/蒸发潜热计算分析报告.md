## 二、问题分析

### 2.1 ⚠️ **严重问题：蒸发速率公式单位不正确**

**当前公式：**
```
m_dot = h_m × (P_sat - P_v) / (R_v × T_surface)
```

**单位分析：**
- `h_m`: m/s
- `(P_sat - P_v)`: Pa = N/m² = kg·m/(s²·m²) = kg/(s²·m)
- `R_v`: J/(kg·K) = m²/(s²·K)
- `T_surface`: K
- 结果单位：m/s × kg/(s²·m) / (m²/(s²·K) × K) = m/s × kg/(s²·m) / (m²/s²) = **kg·s/m³**

**问题：** 正确的质量通量单位应该是 **kg/(m²·s)**，但当前公式得到的是 **kg·s/m³**，单位完全错误！

### 2.2 ⚠️ **问题：蒸发潜热值未考虑温度依赖性**

**当前实现：**
- `L_v = 2.4279e6 J/kg` (固定值)

**实际情况：**
- 0°C: 约 2.50×10⁶ J/kg
- 25°C: 约 2.44×10⁶ J/kg  
- 50°C: 约 2.38×10⁶ J/kg
- 100°C: 约 2.26×10⁶ J/kg

蒸发潜热随温度升高而降低，使用固定值会导致误差，特别是在高温条件下。

### 2.3 ⚠️ **问题：质量传递系数默认值可能不合理**

**当前默认值：** `h_m = 0.01 m/s`

**问题：**
- 质量传递系数应该与对流换热系数相关（通过 Lewis 数）
- 对于自然对流，质量传递系数通常远小于 0.01 m/s
- 对于强制对流（有风），质量传递系数可能更大
- 应该根据对流条件动态计算，而不是使用固定值

### 2.4 ✅ **正确部分**

1. **Tetens 方程**：饱和水蒸气压力计算正确
2. **物理常数**：R_v = 461.5 J/(kg·K) 正确
3. **逻辑判断**：当 P_sat_surface ≤ P_v 时返回 0，逻辑正确

## 三、正确的物理模型

### 3.1 正确的蒸发速率公式

基于质量传递理论，蒸发速率应该使用**水蒸气密度差**：

**方法1：基于密度差（推荐）**
```python
# 水蒸气密度 (kg/m³)
ρ_v_sat = P_sat / (R_v * T_surface)      # 表面饱和水蒸气密度
ρ_v_ambient = P_v / (R_v * T_ambient)    # 环境水蒸气密度

# 质量通量 (kg/(m²·s))
m_dot = h_m * (ρ_v_sat - ρ_v_ambient)
```

**方法2：基于摩尔质量（等效）**
```python
M_w = 18.015e-3  # 水的摩尔质量 (kg/mol)
R = 8.314        # 通用气体常数 (J/(mol·K))

# 质量通量
m_dot = h_m * M_w / (R * T_surface) * (P_sat - P_v)
```

### 3.2 温度相关的蒸发潜热

**经验公式（IAPWS-95 简化）：**
```python
T_C = T_surface_K - 273.15
L_v = (2501.0 - 2.36 * T_C) * 1000  # J/kg
```

或使用更精确的公式：
```python
T_C = T_surface_K - 273.15
if T_C < 0:
    L_v = 2.501e6  # 冰的升华潜热
elif T_C <= 100:
    L_v = (2501.0 - 2.36 * T_C - 0.0016 * T_C**2) * 1000
else:
    L_v = (2257.0 - 0.5 * (T_C - 100)) * 1000
```

### 3.3 质量传递系数的计算

质量传递系数应该与对流换热系数相关：

```python
# Lewis 数（对于水蒸气-空气系统，约等于 1）
Le = 1.0

# 从对流换热系数计算质量传递系数
h_conv = calculate_convection_coefficient(...)  # W/(m²·K)
cp_air = 1005  # 空气比热容 J/(kg·K)
ρ_air = 1.225  # 空气密度 kg/m³（需要根据温度修正）

h_m = h_conv / (ρ_air * cp_air * Le**(2/3))
```

## 五、修正后的代码框架

```python
def _calculate_latent_heat_power(
    T_surface_K: float,
    T_ambient_K: float,
    RH: float,
    h_m: float | None = None,  # 如果为None，则从对流系数计算
    h_conv: float | None = None,  # 对流换热系数
) -> float:
    """
    计算单位面积蒸发潜热功率 (W/m²)
    
    修正后的模型：
    1. 使用水蒸气密度差计算蒸发速率
    2. 考虑温度相关的蒸发潜热
    3. 可选：从对流系数计算质量传递系数
    """
    R_v = 461.5  # 水蒸气气体常数 (J/(kg·K))
    
    # 计算饱和水蒸气压力（Tetens方程）
    T_surface_C = T_surface_K - 273.15
    T_ambient_C = T_ambient_K - 273.15
    
    P_sat_surface = 0.61078 * np.exp(17.27 * T_surface_C / (T_surface_C + 237.3)) * 1000  # Pa
    P_sat_ambient = 0.61078 * np.exp(17.27 * T_ambient_C / (T_ambient_C + 237.3)) * 1000  # Pa
    P_v = RH * P_sat_ambient  # Pa
    
    if P_sat_surface <= P_v:
        return 0.0
    
    # 计算水蒸气密度 (kg/m³)
    ρ_v_sat = P_sat_surface / (R_v * T_surface_K)
    ρ_v_ambient = P_v / (R_v * T_ambient_K)
    
    # 计算质量传递系数（如果未提供）
    if h_m is None:
        if h_conv is None:
            h_m = 0.01  # 默认值（应该改进）
        else:
            # 从对流系数计算
            cp_air = 1005  # J/(kg·K)
            T_film = (T_surface_K + T_ambient_K) / 2
            ρ_air = 1.225 * (273.15 / T_film)  # 理想气体近似
            Le = 1.0  # Lewis数
            h_m = h_conv / (ρ_air * cp_air * Le**(2/3))
    
    # 计算蒸发速率 (kg/(m²·s)) - 修正后的公式
    m_dot = h_m * (ρ_v_sat - ρ_v_ambient)
    m_dot = max(0.0, m_dot)
    
    # 计算温度相关的蒸发潜热 (J/kg)
    if T_surface_C < 0:
        L_v = 2.501e6
    elif T_surface_C <= 100:
        L_v = (2501.0 - 2.36 * T_surface_C - 0.0016 * T_surface_C**2) * 1000
    else:
        L_v = (2257.0 - 0.5 * (T_surface_C - 100)) * 1000
    
    # 计算蒸发潜热功率 (W/m²)
    Q_latent = m_dot * L_v
    
    return float(Q_latent)
```
