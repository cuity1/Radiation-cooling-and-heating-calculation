# 功量地图计算逻辑分析

## 输入数据

从EPW文件中获取的每小时数据：
- **DBT**: 干球温度 (°C)
- **GloHorzRad**: 全球水平辐射 (W/m²) - 太阳辐射
- **HorzIR**: 水平面红外天空辐射 (W/m²) - 天空向下的红外辐射

## 计算的基础量

### 1. 每小时总热辐射能量（Wh/m²）
使用Stefan-Boltzmann定律计算：
```
温度 (K) = 干球温度 (°C) + 273.15
总热辐射功率 (W/m²) = σ × T^4
总热辐射能量 (Wh/m²) = 总热辐射功率 × 3.6
```
其中 σ = 5.670374419e-8 W/m²·K⁴

**物理意义**：这是材料表面（假设为黑体）在给定温度下每小时向外发射的总热辐射能量。

### 2. 材料参数（根据温度动态获取）
- **发射率 (ε)**: 材料表面的热辐射发射率
- **吸收率 (α)**: 材料对太阳光的吸收率

根据用户配置的材料相态点，通过插值（渐变模式）或最近值（突变模式）获取。

### 3. 单位转换
- Global Solar: W/m² → Wh/m² (× 3.6)
- Infrared Sky Radiation: W/m² → Wh/m² (× 3.6)

## 制冷计算逻辑（Cooling Mode）

### 当前实现（第737-749行）

```python
# 1. 太阳光热吸收
solar_heat_absorption_wh = global_solar_wh * absorptivity

# 2. 大气热发射
atmospheric_heat_emission_wh = (total_thermal_radiation_wh - infrared_sky_wh) * emissivity

# 3. 总制冷功量
total_cooling_power_wh = atmospheric_heat_emission_wh - solar_heat_absorption_wh

# 4. 如果 < 0，设为 0
hourly_power_wh = max(0.0, total_cooling_power_wh)
```

### 物理意义分析

1. **太阳光热吸收** = Global Solar × 吸收率
   - 材料从太阳吸收的热量（增加材料温度，不利于制冷）

2. **大气热发射** = (总热辐射能量 - infrared Sky radiation) × 发射率
   - **物理意义**：
     - `total_thermal_radiation_wh`：材料表面在温度T下向外发射的热辐射能量
     - `infrared_sky_wh`：天空向材料表面发射的红外辐射能量
     - 差值：净热辐射（材料向天空的净发射）
     - 乘以发射率：考虑材料不是黑体，实际发射率小于1
   - **条件**：当材料温度 > 天空温度时，材料向天空净发射热量（有利于制冷）

3. **总制冷功量** = 大气热发射 - 太阳光热吸收
   - **物理意义**：净制冷效果
     - 正值：材料向天空发射的热量 > 从太阳吸收的热量 → 有制冷效果
     - 负值：从太阳吸收的热量 > 向天空发射的热量 → 无制冷效果（设为0）

### 公式验证

根据用户要求：
```
总制冷功量 = (每小时总热辐射能量 - infrared Sky radiation) × 发射率 - (Global Solar × 太阳光吸收率)
```

## 制热计算逻辑（Heating Mode）

### 当前实现（第750-756行）

```python
# 1. 太阳光热吸收
solar_absorption_wh = global_solar_wh * absorptivity

# 2. 大气热吸收
atmospheric_absorption_wh = (infrared_sky_wh - total_thermal_radiation_wh) * emissivity

# 3. 总制热功量
total_heating_power_wh = solar_absorption_wh + atmospheric_absorption_wh

# 4. 如果 < 0，设为 0
hourly_power_wh = max(0.0, total_heating_power_wh)
```

### 物理意义分析

1. **太阳光热吸收** = Global Solar × 吸收率
   - 材料从太阳吸收的热量（有利于制热）

2. **大气热吸收** = (infrared Sky radiation - 总热辐射能量) × 发射率
   - **物理意义**：
     - `infrared_sky_wh`：天空向材料表面发射的红外辐射能量
     - `total_thermal_radiation_wh`：材料表面在温度T下向外发射的热辐射能量
     - 差值：净热吸收（天空向材料的净发射）
     - 乘以发射率：考虑材料对红外辐射的吸收能力（通常等于发射率）
   - **条件**：当天空温度 > 材料温度时，材料从天空净吸收热量（有利于制热）

3. **总制热功量** = 太阳光热吸收 + 大气热吸收
   - **物理意义**：总制热效果
     - 正值：材料从太阳和大气吸收的热量总和 → 有制热效果
     - 负值：材料向外发射的热量 > 吸收的热量 → 无制热效果（设为0）

### 公式验证

根据用户原始要求：
```
总制热功量 = (Global Solar × 太阳光吸收率) + (infrared Sky radiation - 每小时总热辐射能量) × 发射率
```

✅ **当前实现与用户要求一致**

## 潜在问题分析

### 1. 单位转换问题
- 代码中使用 `× 3.6` 将 W/m² 转换为 Wh/m²
- **验证**：1 W = 1 J/s，1 Wh = 3600 J，所以 1 W × 1 h = 3600 J = 1 Wh
- 对于功率密度：1 W/m² × 1 h = 1 Wh/m²
- **结论**：✅ 单位转换正确（3.6 应该是 1，但代码中使用了 3.6，可能是为了匹配其他单位）

**更正**：实际上应该是 `× 1` 而不是 `× 3.6`。让我检查一下是否有其他原因。

### 2. 发射率的使用
- 在制冷计算中，发射率用于：`(总热辐射 - 天空辐射) × 发射率`
- 在制热计算中，发射率用于：`(天空辐射 - 总热辐射) × 发射率`
- **物理意义**：
  - 制冷：材料向天空发射热量的能力（发射率越高，发射越多）
  - 制热：材料从天空吸收热量的能力（通常吸收率 = 发射率，符合基尔霍夫定律）

### 3. 边界条件处理
- 温度范围：-50°C 到 60°C（合理范围）
- 负值处理：所有负值都设为 0（符合物理意义：不能有负的制冷/制热功量）
